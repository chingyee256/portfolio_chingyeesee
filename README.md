# Ching Yee See - Product Management Portfolio

A modern, professional portfolio website showcasing experience and qualifications for product management positions.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Smooth Scrolling**: Navigation with smooth scroll effects
- **Interactive Elements**: Hover effects and scroll animations
- **Professional Layout**: Well-organized sections for easy navigation

## Sections

1. **Hero Section**: Introduction with contact information
2. **Career Summary**: Overview of professional background
3. **Work Experience**: Detailed work history including internships and entrepreneurial experience
4. **Education**: Academic qualifications
5. **Honors & Awards**: Achievements and recognitions
6. **Skills**: Technical and language skills
7. **Contact**: Easy-to-use contact information

## How to Use

1. Open `index.html` in a web browser
2. The website is fully functional and doesn't require a server
3. All sections are accessible via the navigation menu

## Customization

- **Colors**: Modify CSS variables in `styles.css` (root section) to change the color scheme
- **Content**: Edit `index.html` to update any information
- **Styling**: Adjust `styles.css` for layout and design changes

## Browser Support

Works on all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features and animations
